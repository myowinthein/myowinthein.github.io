# myowinthein.github.io

Personal landing page for [Myo Win Thein](https://github.com/myowinthein) — lists open source projects with GitHub Pages sites.

## Install

No install required. Clone and open `index.html` in a browser.

```sh
git clone https://github.com/myowinthein/myowinthein.github.io.git
cd myowinthein.github.io
open index.html   # macOS; use xdg-open on Linux or just double-click
```

## Usage

The page fetches the owner's public GitHub repos at runtime and renders cards for any repo that has GitHub Pages enabled. No build step, no server needed.

To deploy changes, push to `main` — GitHub Pages publishes automatically.
